使い方

  1. パソコンを起動してください。

  2. ルートラボのマイページを開きます。

  3. 「あなたが投稿したルート」または、
    「あなたが★をつけたルート」の一括ダウンロードボタンを押します。

  4. ダウンロードしたZIPファイルを解凍します。

  5. フォルダ内の全てのファイルを選択して、ドラッグ＆ドロップします。


ルートラボは2020年3月31日にサービスを終了します。それまでに全てのGPXファイルをダウンロードしておきましょう。
