使い方

  1. ルートラボのマイページを開きます。

  2. 「あなたが投稿したルート」または、
    「あなたが★をつけたルート」の一括ダウンロードボタンを押します。

  3. ダウンロードしたZIPファイルを解凍します。

  4. フォルダ内の全てのファイルを選択して、ドラッグ＆ドロップします。


ルートラボは2020年3月31日にサービスを終了します。それまでに全てのGPXファイルをダウンロードしておきましょう。
